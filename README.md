## 📋 Sobre o Projeto

O **Sistema de Estacionamento Parada Impactante** foi um projeto acadêmico de **Engenharia de Requisitos** desenvolvido pela Faculdade Impacta Tecnologia, focado na modernização completa das operações de estacionamento através da automação de processos.

### 🎯 **Problema Identificado**
O estacionamento "Parada Impactante" enfrentava problemas operacionais devido ao sistema manual:
- ❌ Longas filas na entrada e saída
- ❌ Erros no cálculo do tempo de permanência  
- ❌ Perda frequente de tickets
- ❌ Gestão ineficiente das vagas
- ❌ Insatisfação dos clientes e prejuízos financeiros

### 🚀 **Solução Desenvolvida**
Sistema completo de automação que integra hardware e software para:
- ✅ **90% de redução** nos erros de cálculo
- ✅ **70% mais agilidade** no fluxo de veículos
- ✅ **20% de aumento** no faturamento mensal
- ✅ **Controle em tempo real** da ocupação
- ✅ **Experiência superior** para clientes

## 🏗️ Arquitetura do Sistema Projetada

### **Hardware Especificado**
```
🚪 Controle de Acesso
├── 4 Cancelas automáticas (2 entrada + 2 saída)
├── Leitores de QR Code integrados
├── Câmeras LPR (reconhecimento de placas)
└── Sensores de presença veicular

💳 Sistema de Pagamento  
├── Totens de autoatendimento
├── Leitores de cartão (crédito/débito)
├── Integração PIX
└── Interface touchscreen intuitiva

📊 Monitoramento
├── 150 vagas monitoradas
├── Capacidade: 1.000 veículos/dia
├── Câmeras de segurança 24h
└── Sistema de backup energético
```

### **Software e Funcionalidades Documentadas**

#### 🖥️ **Dashboard Administrativo**
- Monitoramento em tempo real de entrada/saída
- Controle de ocupação das 150 vagas
- Relatórios de faturamento (diário/mensal)
- Gestão de promoções e mensalistas
- Análise de padrões de uso
- Alertas de manutenção

#### 📱 **Interface do Cliente** (Especificada)
- Localização de vagas disponíveis
- Sistema de reservas antecipadas
- Pagamento integrado no app
- Navegação GPS até a vaga
- Histórico de uso e gastos
- Notificações push

## 🛠️ Stack Tecnológica Recomendada

### **Backend**
- **Node.js** com Express.js
- **Python** com Django/FastAPI
- APIs RESTful para integração

### **Frontend**
- **React.js** com TypeScript
- **Vue.js** com Vuetify
- Dashboard responsivo e intuitivo

### **Banco de Dados**
- **PostgreSQL** para dados relacionais
- **MySQL** com replicação
- **Redis** para cache e sessões

### **Infraestrutura**
- **AWS/Azure** para cloud hosting
- **Docker** para containerização
- **CI/CD** com GitHub Actions

### **Hardware Integrado**
- Cancelas automáticas industriais
- Leitores QR Code de alta precisão
- Câmeras LPR com IA integrada
- Gateway IoT para comunicação
- Totens touch resistivos ao tempo
